# Shorten URL Apps
This Application we made for our Final Exam PBO in STMIK Dharma Negara

## Commands

| Command | Description           |
| ------- | --------------------- |
| dev     | Start devserver.      |
| build   | Build for production. |

- [CLSX](https://www.npmjs.com/package/clsx)
